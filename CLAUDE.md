## ワークフロー

### Git について

- **main ブランチに直接コミットすることは禁止**
  - 必ず別ブランチ（`feature/*`, `fix/*`, `docs/*` など）を切ってからコミットすること
  - main ブランチへの変更は必ず PR を通して行うこと
- コミットコメントや PR は全て日本語で書くこと
